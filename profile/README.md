**[Utsuwa](https://www.utsuwa.ai/)**\
An open-source app for creating and animating AI-powered virtual characters, built on the VRM standard. Load a model, connect an LLM, and get a companion with lip sync, expression, and presence. The avatar layer should be portable, inspectable, and owned by the people building with it.
